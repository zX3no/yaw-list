# Yaw-List
Converting sensitivity `Game 1 Sens * Game 1 Yaw รท Game 2 Yaw = Game 2 Sens` 
e.g.: Quake 1.5 sensitvity to Overwatch is: `1.5 * 0.022 รท 0.0066 = 5`

Calculating yaw from [mouse-sensitivity.com](www.mouse-sensitivity.com): `s1 * y1 / s2 = y2`
